Twitter "I love you" Graph
By Akhil Stanilavose
[akhil.stanislavose@gmail.com]
June 19, 2011


Design:
*******
Plain simple, get tweets with "i love you" from twitter using Twitter Search API, plot it to graph!


Dependencies:
*************
PHPTwitterSearch - Wrapper class around the Twitter Search API for PHP.
JQuery - The one and only awesome javascipt library! Big fan!
Highcharts - A javascript charting library.



Usage:
******
Open index.html in your favourite browser!
