Design
	Sandbox application both server and client are developed in java. The application communicates using tcp protocol. Server commands can be executed using client program. The results are shown in both server and client window. Server program can perform function lile listing files,deleting files,reading and creating files.The client program creates new random file in each 1 minute..Type 'Help' in sandbox client program for command list

Dependencies

Install :
	java jdk 1.6

Usage
	First run sandbox_server.jar and then sandbox_client.jar  Source codes are also provided in respective folders.
