DESIGN:

	1. Geting date form the System. here Date function in java.util package is used
	2. Get minutes and hours from the date. Here getHours() and getMinutes() is used.
	3. Each time is checked for its corresponding message. Here an if-else condition statement is used for various conditions.
	4. Display the message. Here paint function is used to display the message. A background image is also used here.

DEPENDENCIES:
	JDK v1.2 or latr is required.

USAGE:
	Just compile the code and run it.The program autimatically gets the system time and displays the corresponding message.
