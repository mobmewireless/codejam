				Contest ID: ternary
1.DESIGN

The problem is implemented in C and consists of three source file: num_gen.c that generates the ten digit numbers, filter.c that filters out the numbers into files on the basis of the first digit of the number, and counter.c, that counts the number count in the files.
	
2. DEPENDENCIES

	2.1 Requires gcc(version 4.5.2) 
	2.2 Requires to link pthread library
		
3. USAGE

	Compile the source files using gcc as gcc <filename> -o <output_filename> -l pthread
	Execute the code in  different terminals as ./<output_filename>
