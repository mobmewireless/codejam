Design:
*******
A simple java class which takes the input zip file and modifies the ODT(OpenDocument Text) files in it using Simple ODF library, also called Simple Java API for ODF, which is an easy-to-use, high-level Java API for creating, modifying and extracting data from Open Document Format (ODF) documents. It is a high level abstraction of the lower-level ODFDOM API.


Dependencies:
*************
1. Simple ODF 0.6. - http://simple.odftoolkit.org/
2. ODFDOM 0.8.7. - http://odftoolkit.org/projects/odfdom/pages/Home
3. The Apache Xerces 2.9.1 or higher version - http://xerces.apache.org/mirrors.cgi

All the dependencies are available in the lib folder.


Usage:
******
Run using java interpreter ie., "java vandal" in console. Enter the source zip file path. Tada!
