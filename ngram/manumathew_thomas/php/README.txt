Design:
		A part of the novel "The Count of Monte Cristo, by Alexandre Dumas, Pere" is stored in story.txt which is imported into the trigram.php file and perform generation of trigrams with makeTrigram() function.
		
		Each trigram is chained with the concat1() function.
		
Dependencies:
			This php can be run on any localsever with php>4.0.
			
Usage:
		To see the ouput just run trigram.php script.
