Design

A novel object is made from the novel file with one of its attributes trigram_list  which contains the
trigrams of the novel.While generating trigrams punctuation marks are retained and it spans newlines. This novel object is then passed to dissociation press class to run dissociation press algorithm on it to generate the short story.


Dependencies

novel file: christmas_carol.txt 


Usage

Place the novel file in the same directory as of the code and run the python code
