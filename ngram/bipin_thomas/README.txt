Design:
A novel to be stored in filename.txt file. The file is imported in the form of string using file_get_contents() function and trigrams are made using makeTrigram() function.Trigrams are chained with the concaten() function.		
			
Dependencies:
			This script can be executed on any localsever with php >4.0.
			
Usage:

Execute the script with any novel in the filename.txt in any local server.
